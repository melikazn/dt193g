Moment 1- DT193G

Den här uppgiften handlar om att bygga en webbplats med två olika CSS-ramverk. Jag valde Bootstrap och Bulma som mina ramverk. Ursprungligen tänkte jag använda Tailwind CSS, men många funktioner i Tailwind kräver en betalversion, så jag bestämde mig för att undvika det. För uppgiften skapade jag en webbplats med fyra olika undersidor. På en av sidorna använde jag Bootstrap som CSS-ramverk. Jag tycker att Bootstrap är mycket enkelt att använda eftersom det erbjuder många färdigbyggda komponenter, där man bara kopierar koden och implementerar den direkt. Det går även att anpassa CSS för att skapa unika webbplatser. En stor fördel med Bootstrap är att det innehåller färdig JavaScript-funktionalitet, vilket sparar mycket tid. Det andra CSS-ramverket jag valde att använda var Bulma. Jag tycker att Bulmas komponenter är snyggare än Bootstraps, och det är också enkelt att använda. En nackdel med Bulma är dock att det saknar inbyggd JavaScript-funktionalitet. Detta innebär att man måste skriva JavaScript själv, vilket kan kräva extra tid.
